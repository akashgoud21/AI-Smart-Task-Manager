# 🤖 AI Smart Task Manager

An AI-powered task management web application developed using **Python Flask, HTML, CSS, and SQLite**. The application helps users efficiently manage tasks, track productivity, prioritize work, and receive intelligent task recommendations.

## 🚀 Features

- User Registration and Login
- Create, Complete, and Delete Tasks
- Priority-Based Task Management
- Deadline Tracking
- Productivity Score Calculation
- AI Task Recommendations
- SQLite Database Integration
- Responsive User Interface

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- SQLite
- Jinja2

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/akashgoud21/AI-Smart-Task-Manager.git
```

### Navigate to the Project Directory

```bash
cd AI-Smart-Task-Manager
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open in Browser

```text
http://127.0.0.1:5000
```
